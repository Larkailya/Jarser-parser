Interface

Your starting position on the browser tab.

There are a navigation bar and actions with parsing.

 

Before beginning any work, you must log in under your account in the instagram.

 

Good. After logging into your account, you can visit any profile and use it as a regular browser. For example, I want to get information about several followers of any profile.

 If in the profile of a very large follower, you can use the "Help Scrolling" button to automatically scroll the page of the followers to the bottom.

After automatic / manual scrolling of the followers, we start parsing. Click on the button “Run parse”.

Parse process started.

After the end of parsing, you will receive a window with the notification of completion.

You can see the results in the Results tab.

You can export the data to the Excel file or clear results.  

In addition, select the fields to export.
